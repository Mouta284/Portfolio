About this site

I decided to build my own portfolio website to test my skills and to create, from scratch, a way to introduce to people my work and side-projects.

In this website you can find projects I've been working on, projects already finished and some other that I'm planning to do in the near future. Since I'm not using any framework, except Bootstrap for the responsiveness of this page, I'll test my own JavaScript habilities and build my own CSS rules, as I've been learning throught my journey at FEUP.

About me

My name is Francisco Mouta, and I'm a passionate developer who likes to idealize projects and make them real. My main goal is to learn even more and practice what I'm being taught in my engineering course at FEUP. My favorite part of computer engineering is building everything from scratch, to have a better knowledge of how things really work behind the curtain.

I think that the best way of gaining experience is through failure and persistence. If we want to succeed, we need to find a way out, with someone's help, or, preferably, by teaching ourselves the way.
